Choose Your Adventure

Addendum

This code defines a text-based choose-your-own-adventure game using JavaScript. The game is structured as a series of interconnected story stages with choices and consequences, leading to different outcomes. Here's an overview of the code:

Story Stages:

The game's narrative is organized into stages, each represented as an object in the storyStages constant. Each stage has the following properties:

  • text: A description of the current situation or narrative at that stage.
  • choices: An array of choices available to the player at that stage.
  • consequences: An array representing the consequences of each choice. The order of choices and consequences should match.
  • image and image2: URLs pointing to images associated with the current stage.

Game Functions:

  • startGame():
    • Resets the game by setting the currentStage to the starting stage ("start").
    • Calls the updatePage() function to display the starting stage on the webpage.
  • updatePage():
    • Retrieves the current stage information based on currentStage.
    • Updates the webpage with the current stage's text and choices.
    • Displays images associated with the current stage.
  • makeChoice(choiceIndex):
    • Handles the player's choice.
    • Retrieves the current stage and determines the next stage based on the player's choice.
    • Updates the currentStage and calls updatePage() to continue the game.

Game Initialization:

  • currentStage: A variable to keep track of the current stage.
  • Calls startGame() to initialize and display the starting stage when the script runs.

HTML Integration:

Assumes there is HTML with elements having the IDs: "story", "choices", "story-image", and "story-image2". These elements are updated dynamically as the game progresses.

How to Play:

  • The player starts at the "start" stage and makes choices to navigate through the story.
  • Choices lead to consequences, which determine the next stage.
  • Images are displayed to enhance the storytelling experience.
